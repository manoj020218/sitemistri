यह कैसे काम करता है?

Simple 6-Step Flow

01
SI
Client Site Location डालें
Map पर pin drop करें या current location use करें। Work type और skill select करें।
02
SI
Nearby Technicians देखें
Available, nearby, और skill-matched technicians की list देखें। Distance, last update, completed work सब दिखेगा।
03
SI + Technician
Call/WhatsApp करें, Discuss करें
Work scope, time, और charges directly बात करके तय करें। App में कोई payment नहीं।
04
SI
Site Work Assign करें
Site details, work type, और agreed time के साथ Site Work create करें।
05
Technician
Accept करें, जाएं, काम करें
Accept करो, site पर जाओ, काम करो, 1 proof photo upload करो।
06
SI
Close करें, Trust बढ़े
SI photo review करके Site Work close करे। Proof photo delete हो जाए। Trust score update हो।
